## Key Concept in API realm
1. API Key
    服务器发的身份凭证, 不能泄露, 自然也不要git add
2. Base URL/Endpoint
    Base URL表示请求发给哪台服务器
3. API格式兼容
    只要消息请求格式一样, 后端是什么模型都一样
4. Model
5. Token 
    API费用: 输入Token数量 * 输入单价 + 输出Token数量 * 输出单价
6. Context Window
    模型单次能看到的最大Token总量
7. Tool Call
8. CLI (Command Line Interface)

## 关于.git/
    **git的提交主要保存的是项目快照之间的对应关系**
    **git的核心可以看作是一个由哈希寻址的对象数据库**
    - git的四种对象
        - blob
            保存文件内容, 不关心文件名
        - tree
            保存目录结构和文件名
        - commit
            保存一次提交的信息
            记录内容如下所示:
                tree <根目录快照>
                parent <父提交>
                author <作者>
                committer <提交者>
                提交说明
        - tag
    - .git/ 的文件目录
    ``` plain_text
        .git/
        ├── COMMIT_EDITMSG
        ├── FETCH_HEAD
        ├── HEAD
        ├── ORIG_HEAD
        ├── branches
        ├── config
        ├── description
        ├── hooks
        ├── index
        ├── info
        ├── logs
        ├── objects
        ├── packed-refs
        └── refs
    ```

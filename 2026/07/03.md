tar 用法
tar is a program designed to store multiple files in a single file ( an archive ), and to manipulate such file.

1. traditional style
    tar + a cluster of options + arguments needed + non-option arguments ( files or archive member names )
    eg. 
    ```
    tar cfv etc.tar /etc
    ```
    ** 必需的参数的顺序要和options的顺序一致 **

2. unix or short-option style
    ** 如果有选项需要参数, 后面一定要加空格跟着, 如果有可选参数, 就一定要不加空格直接写 **
    ** 不需要参数的选项可以并在一起写, 有参数的也可以并, 但是只能放在最后一个 **
    eg. 
    ``` 
    tar -cvf etc.tar /etc
    ```

3. GNU or long-option style
    ** 可以简写 **
    有三种加参数的方式: 
    - 空格 + command line word
    - 无空格直接加在后面
    - =参数

    eg.
    ```
    tar --create --file etc.tar --verbose /etc
    tar --cre --file=etc.tar --verb /etc
    ```

tar 的运行模式
** the meaning of non-option arguments depends on the mode **
- -A, --catenate, --concatenate
    append archives to the end of another archive
    non-option arguments are treated as the names of archives to append
- -c, --create
    create a new archive
    non-option arguments are the names of files to be archives
- -d, -diff, --compare
    find the differences between archive and file system
    non-option arguments is the archive members to be compared, if not given, compare all with the current directory
    ** 但是如果本地有新增的文件, tar是不知道的, 所以它只能比较tar中的文件和指定目录下(默认本目录, 可以通过)下的文件有没有不同 **
- --delete
    delete from the archive
    non-option argumnets are the archive members to be removed
- -r, --append
    append files to the end of an archive
    non-option arguments are the files to be arvhives
- -t, --list
    list the contents of an archive
    non-option arguments are the members to list, if not given, default is all
- --test-label
    test the archive volume label and exit
    ** volune label 卷包, 是给这个archive起的内部名字, 外部显示在文件系统中的名字可以来回变, 但是内部名字不变, 可以在创建archive的时候给, 用-V, --label=TEXT **
- -u, --update
    append files which are newer than the corresponding copy in the archive
    non-option arguments are the members to be updated
- -x, --extract, --get
    extract files from an archive
    non-option arguments are the members to be extracted

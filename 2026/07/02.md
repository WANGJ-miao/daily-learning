# csapp第一天:
- makefile 用法

Makefile fundamental structure:

```
target: dependences
    command
```
** command 前面必须是TAB **
** 这里的dependences用处是和target的修改时间比较, 用于决定是否要重新编译, 并非编译层面的依赖 **
举个例子:
```
btest: btest.c bits.c decl.c tests.c btest.h bits.h
	$(CC) $(CFLAGS) $(LIBS) -o btest bits.c btest.c decl.c tests.c
```

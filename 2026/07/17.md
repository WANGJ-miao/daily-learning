## Makefile
一般Makefile会分成编译和链接两个阶段写
``` Makefile
app: main.o func.o
    gcc main.o func.o -o app

main.o: main.c
    gcc -c main.c -o main.o #-c表示只编译

func.o: func.c
    gcc -c func.c -o func.o 
```
### Makefile 中定义和使用变量
- 定义:
GXX = gcc
TARGET = app
CXXFLAGS = -std=c++20 -Wall -Wextra
- 使用
app: main.o foo.o
    $(GXX) main.o foo.o -o $(Target)
